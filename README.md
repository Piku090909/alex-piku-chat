# 𝛥𝐿𝛯𝛲-𝛲𝛪𝛫𝛥 Web Messaging Platform

A high-fidelity, feature-rich WhatsApp-style communication platform built using React and Tailwind CSS. The application supports secure authentication, private instant messaging, multi-functional group creation via sharable links, custom media distribution, native browser voice recordings, and an advanced Owner Dashboard for absolute system control.

## 🚀 Key Features

*   **🔒 Secured Credentials & Guarded Entry:** Owner credentials are fully masked from clients and managed safely inside the core authentication flow.
*   **🤖 Gemini 3.5 Pro Integration:** A dedicated, state-of-the-art AI assistant built into the chat ecosystem capable of parsing user queries and responding to multi-modal file distributions.
*   **📁 Full Media Delivery Network:** Send and render Photos, Videos, Documents, and Music files (.mp3) instantly in Private, Group, and AI contexts using optimized browser-side storage.
*   **🎙️ Native Voice Notes:** Built-in microphone recording architecture utilizing the browser's HTML5 `MediaRecorder` API to seamlessly record and distribute voice files.
*   **👥 Fixed Group Architecture & Share Links:** Create chat groups on the fly and share dynamically generated invite tokens to add any user instantly.
*   **🛠️ Owner Command Center:** Complete real-time system management. The site owner can switch global themes, toggle platform features, wipe histories, and delete any message system-wide.
